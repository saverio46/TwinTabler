Upload Your Excel File

How It Works:

  1. Upload an Excel file (.xlsx) containing customer data.
  2. The first column should contain numeric IDs to determine odd/even rows.
  3. The application will process your file:
    • Odd-ID rows will be placed starting from Column A.
    • Even-ID rows will be placed starting from Column G.
    • Headers for even-ID data will have "_2" appended.
  4. The processed file will be available for download automatically.
Only .xlsx files are accepted
